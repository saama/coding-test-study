# 문제 해설

이 디렉토리는 해결한 문제들의 상세한 해설을 저장하는 공간입니다.

## 파일 명명 규칙

### 백준 (Baekjoon)
- `boj-{문제번호}-{문제제목}.md`
- 예: `boj-1000-a-plus-b.md`

### 프로그래머스 (Programmers)
- `prog-{레벨}-{문제제목}.md`
- 예: `prog-level2-타겟넘버.md`

### 리트코드 (LeetCode)
- `leet-{문제번호}-{문제제목}.md`
- 예: `leet-1-two-sum.md`

## 해설 작성 형식

```markdown
# [플랫폼] 문제제목

## 문제 링크
- [문제 바로가기](링크)

## 문제 요약
간단한 문제 요약

## 접근 방법
1. 문제 분석
2. 알고리즘 선택 이유
3. 시간/공간 복잡도 분석

## 핵심 아이디어
문제 해결의 핵심 아이디어

## 코드
\`\`\`java
// 코드 구현
\`\`\`

## 배운 점
문제를 통해 배운 점이나 주의사항
```
