# 스터디 계획

이 디렉토리는 코딩테스트 준비를 위한 학습 계획과 진도를 관리하는 공간입니다.

## 추천 학습 순서

### 1단계: 기초 알고리즘 (1-2주)
- [ ] 기본 입출력
- [ ] 배열과 리스트
- [ ] 문자열 처리
- [ ] 기본 수학 연산

### 2단계: 핵심 알고리즘 (3-4주)
- [ ] 정렬 알고리즘
- [ ] 이진 탐색
- [ ] 재귀와 백트래킹
- [ ] 스택과 큐

### 3단계: 그래프 알고리즘 (3-4주)
- [ ] DFS/BFS
- [ ] 최단 경로 (다익스트라, 플로이드-워셜)
- [ ] 최소 신장 트리
- [ ] 위상 정렬

### 4단계: 동적 프로그래밍 (3-4주)
- [ ] DP 기본 개념
- [ ] 1차원 DP
- [ ] 2차원 DP
- [ ] 최적화 기법

### 5단계: 고급 알고리즘 (4-6주)
- [ ] 트리 알고리즘
- [ ] 분할 정복
- [ ] 그리디 알고리즘
- [ ] 세그먼트 트리
